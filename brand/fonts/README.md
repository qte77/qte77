# Fonts

Font binaries are gitignored. Run `uv run scripts/install_fonts.py` to
download the required files into this directory.

## Required files

- `Inter-Bold.ttf`
- `Inter-Regular.ttf`
- `JetBrainsMono-Regular.ttf`
- `JetBrainsMono-Bold.ttf`

Sourced from the [Fontsource](https://fontsource.org/) CDN (jsDelivr-hosted),
which redistributes the upstream OFL fonts as TTF.
