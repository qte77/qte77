# Icon set

Third-party **brand and technology logos**, used nominatively to identify the tools they name
(in the profile README) or to link out to a project. Each mark remains the trademark of its
owner; inclusion here does not imply endorsement.

## `github-black.svg` / `github-white.svg`

- **Shape / code** — the 16px GitHub mark path from
  [Bootstrap Icons](https://github.com/twbs/icons), **MIT** (© The Bootstrap Authors). The
  SVG files are free to use, modify, and redistribute under that licence.
- **Trademark** — the GitHub logo (Octocat / Invertocat) is a trademark of GitHub, Inc. Per
  [GitHub's brand guidelines](https://brand.github.com/foundations/logo) it may be used to
  **link to** a GitHub project but **must not be recoloured**. These ship in GitHub-permitted
  **black (`#181717`)** and **white** only — never an EyeRest tint — paired light/dark like
  the other monochrome marks, and consumed by
  [`../../../brand/ui-kit/github-links.html`](../../../brand/ui-kit/github-links.html).

## Other marks

`claude.svg`, `huggingface.svg`, `astral-*.svg`, `gh-models-*.svg`, `devcontainers-*.svg`,
`ros-*.svg`, `mcp-*.svg`, and `markdown-*.svg` are the respective trademarks of their owners, used
nominatively to identify each technology (`claude.svg`, `astral-*.svg`, `mcp-*.svg`, and
`markdown-*.svg` use the CC0 mark paths from
[Simple Icons](https://github.com/simple-icons/simple-icons)). Brand-colored marks (`claude.svg`
`#D97757`) ship as a single file in the brand color; monochrome-by-brand marks — `mcp-*` and
`markdown-*` (both `#000000`), like `gh-models-*` / `ros-*` / `devcontainers-*` — ship paired
light/dark. `astral-*.svg` (Astral, covering the uv + ruff toolchain) ships paired in the Astral
brand duotone — neon `#D7FF64` on dark, brand-dark `#261230` on light. Use them unmodified and only
to refer to the tool they name.
