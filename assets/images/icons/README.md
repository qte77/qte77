# Icon set

Third-party **brand and technology logos**, used nominatively to identify the tools they name
(in the profile README) or to link out to a project. Each mark remains the trademark of its
owner; inclusion here does not imply endorsement.

## `github.svg`

- **Shape / code** — the 16px GitHub mark path from
  [Bootstrap Icons](https://github.com/twbs/icons), **MIT** (© The Bootstrap Authors). The
  SVG file is free to use, modify, and redistribute under that licence.
- **Trademark** — the GitHub logo (Octocat / Invertocat) is a trademark of GitHub, Inc. Per
  [GitHub's brand guidelines](https://brand.github.com/foundations/logo) it may be used to
  **link to** a GitHub project but **must not be recoloured**. It is therefore rendered only
  in GitHub-permitted **black (`#181717`) / white** — never an EyeRest tint — by
  [`../../../brand/ui-kit/github-links.html`](../../../brand/ui-kit/github-links.html).

## Other marks

`claude.svg`, `huggingface.svg`, `gh-models-*.svg`, `devcontainers-*.svg`, and `ros-*.svg` are
the respective trademarks of their owners, used nominatively to identify each technology. Use
them unmodified and only to refer to the tool they name.
